# AI Mock Interview

AI Mock Interview is an Android application built using Java and Firebase that helps users prepare for job interviews through AI-generated questions and performance evaluation.

## Features

* User Registration and Login
* Interview Category Selection
* AI-Generated Interview Questions
* Performance Evaluation and Feedback
* Interview History Tracking
* User Profile Management
* Firebase Authentication
* Firebase Realtime Database Integration

## Tech Stack

* Java
* Android Studio
* Firebase Authentication
* Firebase Realtime Database
* Gemini API
* XML

## Project Structure

* Authentication Module
* Home Dashboard
* Interview Question Module
* Evaluation Module
* History Module
* Profile Module

## How to Run

1. Clone the repository.
2. Open the project in Android Studio.
3. Connect Firebase to the project.
4. Add the required API keys and configuration files.
5. Build and run the application on an Android device or emulator.
